You're starting from scratch - perfect! We'll use Claude Code to build your React Native app from the ground up.

What You'll Need

  • Claude Pro account - for building your app
  • An app idea - what do you want to build?
  • Basic requirements - main features you want

Step 1: Install Node.js

First, make sure you have Node.js installed:

node --version

If not installed, download from nodejs.org (get the LTS version).

Step 2: Install Claude Code

Claude Code is a command-line tool that helps you build apps with AI assistance.

npm install -g @anthropic-ai/claude-code

üí° What is Claude Code?

Claude Code is an AI coding assistant that runs in your terminal. It can create entire apps, write code files, and help you build your project from scratch.

Step 3: Create Your App with Claude Code

Now use Claude Code to build your app. Here's how:

  1. Open your terminal
  2. Navigate to where you want your project
  3. Start Claude Code:
claude-code

Then give Claude a detailed prompt about your app:

üìù Example Prompt

"Create a React Native app with Expo that [describe your app]. The app should have [list main features]. Use a clean, modern design with [your color preferences]. Make sure it works on iOS."

Be Specific About:

  • What your app does
  • Main features and screens
  • Design preferences (colors, style)
  • Any specific functionality you need

Step 4: Test Your App

Once Claude Code creates your app:

  1. Navigate to your project folder
  2. Install dependencies: npm install
  3. Start the development server: npx expo start
  4. Download Expo Go on your phone
  5. Scan the QR code to test your app

Step 5: Iterate and Improve

If something isn't working or you want to add features:

  1. Go back to Claude Code
  2. Describe what you want to change
  3. Claude will modify the code
  4. Test again in Expo Go

‚úÖ When It Works

Once your app runs smoothly on your phone through Expo Go, you're ready to move to the next module where we'll configure it for the App Store.

Common Questions

How long does this take?

Simple apps: 30 minutes to 1 hour. Complex apps: 2-4 hours of iterating with Claude.

What if I get errors?

Copy the error message and ask Claude Code to fix it. It's very good at debugging.

Can I change things later?

Absolutely! You can keep improving your app even after it's on the App Store.
