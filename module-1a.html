Perfect! Since you already have a React Native app built with Expo, we just need to set up the tools to deploy it to the App Store.

What You'll Install

  • Node.js and npm (if not already installed)
  • Expo CLI
  • EAS CLI (for building)
  • Xcode Command Line Tools (Mac only)

Step 1: Check Node.js

Since you used Replit, Node.js might already be installed. Let's check:

node --version

If you see a version number (like v20.x.x), you're good! Skip to Step 2.

If not installed, download from nodejs.org and install the LTS version.

üí° Tip

Make sure you have Node.js version 18 or higher.

Step 2: Install Expo CLI

This lets you run Expo commands from your terminal.

npm install -g expo-cli

Verify it works:

expo --version

Step 3: Install EAS CLI

EAS CLI creates production builds for the App Store.

npm install -g eas-cli

Verify:

eas --version

Step 4: Xcode Command Line Tools (Mac Only)

If you're on a Mac, you need these tools:

xcode-select --install

Click "Install" when the dialog appears. This takes 10-15 minutes.

‚ö†Ô∏è Windows/Linux Users

Skip this step. You'll use EAS Build's cloud service instead (covered in Module 4).

Step 5: Create Expo Account

Sign up at expo.dev/signup, then log in:

eas login

Verify Everything

Run these commands to check your setup:

node --version
npm --version
expo --version
eas --version
eas whoami

If all return version numbers or your username, you're ready!

‚úÖ Checkpoint

You now have all the tools installed to deploy your React Native app to the App Store.

Common Issues

Permission errors

Try using sudo:

sudo npm install -g expo-cli

Command not found

Restart your terminal and try again.
