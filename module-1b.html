You have a web app that needs to be converted to React Native for iOS. This path will guide you through the conversion process.

‚ö†Ô∏è Important Note

Converting a web app to React Native is complex and may require significant code changes. Some web-specific features won't work on mobile and will need alternatives.

Understanding the Difference

Web apps and mobile apps work differently:

  • Web: Uses HTML, CSS, runs in browsers
  • React Native: Uses React components, runs as native iOS app

Conversion Strategy

We'll use Claude to help convert your web app. Here's the approach:

Step 1: Assess Your Web App

First, understand what you have:

  • What framework? (Plain HTML, React, Next.js, etc.)
  • Key features and functionality
  • External dependencies or APIs

Step 2: Set Up React Native Project

Create a new React Native project with Expo:

npx create-expo-app my-app
cd my-app

Step 3: Use Claude Code to Convert

Open Claude Code in your terminal and ask it to help convert your web code to React Native:

üí° Example Prompt

"I have a web app built with [your framework]. I need to convert it to React Native with Expo. Here's my main component: [paste code]. Can you help me convert this to React Native?"

Common Conversions

HTML ‚Üí React Native Components

  • <div> ‚Üí <View>
  • <span>, <p> ‚Üí <Text>
  • <button> ‚Üí <Button> or <TouchableOpacity>
  • <input> ‚Üí <TextInput>
  • <img> ‚Üí <Image>

CSS ‚Üí StyleSheet

React Native doesn't use CSS files. Instead, you use StyleSheet objects:

const styles = StyleSheet.create({
container: {
flex: 1,
backgroundColor: '#fff'
}
});

Testing Your Conversion

Once Claude helps you convert the code:

  1. Install dependencies: npm install
  2. Start the dev server: npx expo start
  3. Test on your phone with Expo Go
  4. Fix any errors with Claude's help

‚úÖ When It Works

Once your app runs smoothly in Expo Go on your phone, you're ready to move to the next module where we'll configure it for the App Store.

Need Help?

If you get stuck during conversion, ask Claude Code specific questions about:

  • Error messages you're seeing
  • Specific features that aren't working
  • How to implement mobile-specific alternatives
